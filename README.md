# 電商網站 : SweetHome 法式甜點

## 使用技術 :
- Vue3 框架
- Vue CLI建立專案
- 使用 AJAX 串接 RESTful API
- JavaScript 原生語法應用
- Bootstrap 5 + SCSS 建立網頁並規劃 RWD 版型
- Swiper、 vee-validate、Sweetalert2、vue-loading-overlay 套件使用

## 電商網頁功能 :
- 商品分類切換
- 單一產品展示
- 搜尋產品功能
- 加入我的最愛功能
- 完整購物車結帳流程
